In agentic AI built in tools and libraries are utilized to make different calculations and processing fast. We can hardcode some common problems and pass them to the model to use them.

Click on get started to navigate to the project page

Here you can upload data and see the insights and visuals.

Click on upload.

Select your data to be displayed on data table section.

Click on clean to initiate data cleaning.

Similary you can analyze and the prompt to analyze the data will be passed on to the LLM.

Finally the autogluon library will build some models on its own and will produce the results. The LLM will analyze the data and the model evaluations to give insights about the performance of the models and what else can be done to increase their performance.
